== Description ==

[Português]
Adiciona um filtro para páginas filhas no painel administrativo do WordPress, facilitando a gestão de páginas hierárquicas.

[English]
Adds a parent page filter to the WordPress admin panel, making it easier to manage hierarchical pages.

**Key Features:**

* Simple dropdown filter in the pages list
* Filter pages by parent page
* Zero configuration needed
* Performance optimized
* Works with any theme

**Perfect for:**

* Sites with many nested pages
* Complex page hierarchies
* Educational websites
* Documentation sites
* Any WordPress site using hierarchical pages

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/page-parent-filter` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to Pages > All Pages to use the filter

== Frequently Asked Questions ==

= Does this plugin modify my database? =

No, this plugin only adds a filter to the admin interface and doesn't modify any content or structure in your database.

= Will this work with custom post types? =

Currently, this plugin is designed to work specifically with WordPress pages.

= Is this plugin compatible with my theme? =

Yes, this plugin works independently of your theme and should be compatible with any WordPress theme.

= Does it support multisite? =

Yes, the plugin is fully compatible with WordPress multisite installations.
